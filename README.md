## ✨ Features

### 🎨 Visual Excellence
- **3D Interactive Vehicle** - Rotating wireframe model with orbiting particles
- **Dynamic Cursor Spotlight** - Interactive spotlight that follows mouse movements
- **Glassmorphism Design** - Modern frosted glass effects with blur backgrounds
- **Animated Gradients** - Fluid color transitions and gradient text effects
- **Custom Noise Texture** - Subtle grain texture for premium feel

### 🚀 Interactive Components
- **Theme Toggle** - Smooth dark/light mode switching with sun/moon animation
- **Sticky Navigation** - Smart header that transforms on scroll with section highlighting
- **Count-Up Statistics** - Animated counters that trigger on viewport entry
- **Horizontal Scrolling Cards** - Premium services showcase with 3D tilt on hover
- **Interactive Tabs** - Dynamic technology showcase with animated diagrams
- **Mobility Experience Selector** - Choose between SDV, AI, or EV to transform page content
- **Live Dashboard** - Real-time metrics with animated bar charts

### 📱 Responsive Design
- Fully responsive across mobile, tablet, and desktop
- Mobile-optimized hamburger menu with smooth animations
- Touch-friendly interactive elements
- Adaptive typography and spacing

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

    # Clone the repository
    git clone https://github.com/yourusername/vaahan-landing-page.git
    
    # Navigate to project
    cd vaahan-landing-page
    
    # Install dependencies
    npm install
    
    # Start development server
    npm run dev
    
    # Build for production
    npm run build

Environment Variables
No environment variables required - runs out of the box!
